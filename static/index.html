Idena Whitelist API

This server provides endpoints to build and download strict whitelists from your local Idena node.

Example usage

# trigger a new snapshot
curl http://localhost:8081/snapshot

# download the latest whitelist
curl -L http://localhost:8081/whitelist/download -o whitelist.jsonl

# check a single address
curl "http://localhost:8081/whitelist/check?address=0x..."

The IDENA_RPC_KEY environment variable must be set if your node requires an API key.

You can replace this page by placing your own index.html in the static/ directory.
